# Employee Turnover Analytics

This project predicts employee attrition using machine learning.

## Tools Used
- Python
- Pandas
- Scikit-learn
- SMOTE
- K-Means Clustering

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Objective
To identify employees likely to leave the company and suggest retention strategies.

## Dataset
HR Analytics dataset containing employee satisfaction, evaluation scores, salary levels and promotions.

## Results
Random Forest achieved the best performance in predicting employee turnover.
